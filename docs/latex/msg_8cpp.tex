\doxysubsection{Descripción detallada}
Funciones de diagnóstico para verificar la configuración SPI del Arduino Mega 2560. 

Este archivo implementa herramientas de depuración que permiten comprobar si los pines SPI están configurados correctamente tanto en modo Maestro como en modo Esclavo. También incluye una función para mostrar el estado del registro SPCR.

\begin{DoxyNote}{Nota}
Estas funciones no afectan la comunicación SPI; solo muestran información útil para validar wiring, modos y configuración del hardware. 
\end{DoxyNote}

Muestra el estado del registro SPCR (SPI Control Register). 

Muestra el contenido del registro SPCR para depuración.

Desglosa cada bit del registro SPCR e imprime su significado:\+
\begin{DoxyItemize}
\item SPIE:\+ interrupción SPI habilitada/\+deshabilitada ~\newline

\item SPE:\+ SPI habilitado/\+deshabilitado ~\newline

\item DORD:\+ orden de bits (LSB/\+\+MSB) ~\newline

\item MSTR:\+ modo maestro/\+esclavo ~\newline

\item CPOL:\+ polaridad del reloj ~\newline

\item CPHA:\+ fase del reloj ~\newline

\item SPR1/\+\+SPR0:\+ divisor de frecuencia ~\newline

\end{DoxyItemize}

También calcula y muestra la velocidad SPI resultante.
