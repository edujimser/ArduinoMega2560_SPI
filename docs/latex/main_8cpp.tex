\doxysubsection{Descripción detallada}
Lógica principal del sistema SPI para el Arduino Mega 2560. 

Este archivo implementa la inicialización del sistema según el modo seleccionado (MAESTRO, ESCLAVO o ESCLAVO\+\_\+\+PRUEBA), el bucle principal de ejecución y las rutinas de interrupción asociadas al SPI y al pin SS.

Maneja:\+
\begin{DoxyItemize}
\item Recepción de paquetes SPI
\item Procesamiento de comandos
\item Validación CRC8 y checksum
\item Respuesta automática en modo SLAVE\+\_\+\+PRUEBA 
\end{DoxyItemize}
