Descripción detallada

Funciones de diagnóstico y verificación de la configuración SPI en el Arduino Mega 2560.

Este archivo declara funciones utilizadas para comprobar el estado de los pines SPI, validar la configuración del registro SPCR y mostrar información útil para depuración.

Nota
Forma parte del sistema de comunicación SPI entre Maestro y Esclavo.

◆ mostrar_configuracion_SPCR()

void mostrar_configuracion_SPCR ( )

Muestra el contenido del registro SPCR para depuración.

Imprime en el monitor serie los bits relevantes del registro SPCR, permitiendo verificar la configuración del módulo SPI.

Muestra el contenido del registro SPCR para depuración.

Desglosa cada bit del registro SPCR e imprime su significado:

  • SPIE: interrupción SPI habilitada/deshabilitada
  • SPE: SPI habilitado/deshabilitado
  • DORD: orden de bits (LSB/MSB)
  • MSTR: modo maestro/esclavo
  • CPOL: polaridad del reloj
  • CPHA: fase del reloj
  • SPR1/SPR0: divisor de frecuencia

También calcula y muestra la velocidad SPI resultante.

bool verificar_configuracion_spi_PIN_Master ( )

Verifica si la configuración de pines SPI corresponde al modo Maestro.

Comprueba que los pines MOSI, MISO, SCK y SS están configurados correctamente para operar como Maestro SPI.

Devuelve
true si la configuración es correcta, false en caso contrario.

Verifica si la configuración de pines SPI corresponde al modo Maestro.

Comprueba:

  • MOSI (51) → OUTPUT
  • MISO (50) → INPUT
  • SCK (52) → OUTPUT
  • SS (53) → OUTPUT y en HIGH
Devuelve
true si todos los pines están configurados correctamente, false si se detecta algún error.
