Descripción detallada

Lógica principal del sistema SPI para el Arduino Mega 2560.

Este archivo implementa la inicialización del sistema según el modo seleccionado (MAESTRO, ESCLAVO o ESCLAVO_PRUEBA), el bucle principal de ejecución y las rutinas de interrupción asociadas al SPI y al pin SS.

Maneja:

  • Recepción de paquetes SPI
  • Procesamiento de comandos
  • Validación CRC8 y checksum
  • Respuesta automática en modo SLAVE_PRUEBA
